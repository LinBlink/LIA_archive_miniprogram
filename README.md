## 目录结构

```
lia-archive/
├── app.js              # 全局入口 + 微信登录 + 请求封装
├── app.json            # 全局配置（tabBar、页面路由）
├── app.wxss            # 全局样式（设计 token）
├── utils/
│   └── util.js         # 工具函数（格式化、字段处理）
└── pages/
    ├── index/          # 首页：档案列表 + 语言过滤
    ├── detail/         # 详情页：档案内容 + 讨论区 + 收藏/举报
    ├── search/         # 检索页：关键词 + 动态热门标签 + 地区分布
    ├── profile/        # 个人中心：登录/登出 + 语言偏好设置
    ├── login/          # 微信一键登录页
    ├── create/         # 档案录入页（含草稿功能）
    ├── my-archives/    # 我的档案列表
    └── my-collects/    # 我的收藏列表
```

## 功能概览

| 功能 | 说明 |
|------|------|
| 档案浏览 | 支持按类型（民间/官方/第三方）、状态（调查中/完结）、语言过滤 |
| 语言过滤 | 用户设置语言偏好（全部/中文/英文），全局持久化 |
| 动态检索 | 关键词搜索 + 热门标签（来自后端）+ 按地区浏览（动态数据） |
| 档案详情 | 正文、人物关系、时间线、证据链、参考来源、讨论区 |
| 用户登录 | 微信一键登录（wx.getUserProfile + wx.login），无需单独注册 |
| 档案录入 | 表单投稿，支持人物/时间线/参考链接动态填写，含草稿保存 |
| 收藏功能 | 收藏/取消收藏档案，持久化到后端 |
| 讨论区 | 档案详情页内嵌评论，支持分页加载 |
| 举报功能 | 对违规/错误档案提交举报 |
| 我的档案 | 查看并删除自己录入的档案 |
| 我的收藏 | 查看并取消收藏的档案 |
| 草稿箱 | 本地草稿存储，下次打开录入页时恢复 |

## 设计系统

| Token | 值 | 用途 |
|-------|----|------|
| bg-primary | `#0A0D0F` | 页面底色 |
| bg-card | `#0F1318` | 卡片底色 |
| accent-red | `#C8392B` | 警示色、官方档案 |
| accent-green | `#8dffdf` | 系统状态、标签 |
| text-main | `#E8DCC8` | 主文字（泛黄纸张感）|
| text-muted | `#6B7888` | 次要文字 |
| border | `#1F2832` | 分割线 |

字体：`'Courier New', monospace`（编号/标签）+ 系统默认（正文）

## 快速开始

1. 将整个目录导入微信开发者工具
2. 在 `app.js` 中将 `baseUrl` 替换为实际后端地址
3. 小程序管理后台配置合法域名
4. 准备 tabBar 所需的 6 张图标（`images/` 目录）
5. 若暂无后端，首页和详情页内置了 mock 数据可直接预览

## 字段映射

```js
// type 字段
0 → 民间档案（灰蓝色条）
1 → 官方档案（警示红色条）
2 → 第三方档案（警戒橙色条）

// status 字段
0 → 未结案（红色脉冲动画）
1 → 已结案（静态灰色）

// lang 字段
0 → 中文
1 → English

// is_private 字段
0 → 公开
1 → 私密（仅投稿人可见）
```

### characters 结构
```json
[{ "id": 1, "name": "张某", "role": "证人", "desc": "说明", "color": "#C8392B" }]
```

### timelines 结构
```json
[{ "time": "2024-01-01 / 02:00", "event": "事件标题", "detail": "详情" }]
```

### evidence 结构
```json
[{ "title": "证据标题", "desc": "说明", "tags": ["标签"], "credibility": "★★★" }]
```

### ref_links 结构
```json
[{ "title": "来源标题", "url": "https://..." }]
```
